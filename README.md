# BaysWorld v2.0

Personal project portfolio & developer knowledge base.  
A lightweight, zero-config app — no Docker, no Redis, no database server needed.

## Quick Start

```bash
cd api
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- **Project Portfolio** — Track all your projects with tech stacks, status, links
- **Knowledge Base** — Organize notes by category (Cyber Security, DevOps, etc.)
- **Markdown Support** — Write notes in Markdown with code block rendering
- **Search** — Global search across projects and notes
- **Zero Config** — JSON file storage, no external dependencies

## Tech Stack

- **Backend**: Node.js, Express
- **Frontend**: Vanilla HTML/CSS/JS (SPA)
- **Storage**: JSON files (in `data/` folder)
